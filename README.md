# vue-frame

> 基于vue2.0+的脚手架  

## 技术栈
vue2 + vuex + vue-router + axios + mockjs + less + ES6 + elementUI2

## 改动说明
- [x] 关闭build时的SourceMap
- [x] vue,vuex,vue-router,axios采用CDN引入，不打包，减少体积
- [x] 开发环境下引入mockjs  
- [x] 引入element-ui  
- [x] 添加支持JSX
